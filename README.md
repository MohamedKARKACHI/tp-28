# Projet Java avec Scripts de Déploiement

Application Java avec scripts de déploiement automatisé.

## Structure

- `simple-java-project/` - Application Java
- `scripts/` - Scripts de déploiement

## Compilation

```bash
cd simple-java-project
mvn clean package
```

## Déploiement

```bash
./scripts/deploy.sh
```
